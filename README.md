# VIVI

## Prerequisites
```
ubuntu server 20.04
```

## Configuration
```
git clone https://github.com/vivitek/box.git
wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
bash
nvm use 14
npm i
node create-openvvrt.js
```
